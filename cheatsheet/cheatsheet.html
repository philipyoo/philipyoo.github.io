COMMAND Info Example
SELECT data type or column in table region_name
FROM table name regions
INNER JOIN join other table states
ON continuing the join regions.id = states.region_id
WHERE conditional statement region_id = 3
BETWEEN x AND y conditional region_id BETWEEN 2 AND 4
LIKE search for specific pattern state_name LIKE "%aliforni%"
AND / OR conditional WHERE region_id = 3 AND state_name LIKE "%ar%"
ORDER BY order data region_id
DESC or ASC descending or ascending ORDER BY region_id DESC

CSS (Cascading Style Sheets)

positioning
- static = default positioning
- relative = positioned relative to its normal position
- fixed = positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled
- absolute = positioned relative to the nearest positioned ancestor
