Title: What is Active Record?

Date: July 19, 2015

Previous Post | Back to Index | Next Post

Model-View-Controller (MVC) is a software architectural design where it splits roles into 3 sections. The model is the storage and use of data. The view is the visual appearance of the software. The controller can be seen as the middleman between the user and the users inputs with the software and its model and view.

Active Record is what Rails uses for its model portion of the MVC. What Active Record does is connect the Ruby objects to relational database tables, and therefore Active Record is essentially an object-relational map (ORM).

An ORM is simply a programming technique which helps convert incompatible types to types that can be used. Active Record makes every Rails developers lives easier by making translations between Ruby objects and the database.

Active Record is an essential tool that saves a developer time, decreases lines of code required, and makes Rails the great tool that it is. Looking forward to Rails!!
